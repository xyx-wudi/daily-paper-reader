## 摘要
为了提升强化学习在现实世界中的适用性，对抗鲁棒强化学习领域研究如何在对抗性环境扰动下训练智能体。在该设定中，主角智能体在面对对手的环境扰动时优化策略，形成零和马尔可夫博弈。当对抗鲁棒强化学习与基于模型的强化学习结合时，对手可以针对学到的转移模型而非训练环境进行攻击。基于此思想，本文引入了深度强化学习智能体在推理时的后验鲁棒化方法。通过将学到的模型与训练好的标称策略结合，我们的方法执行了鲁棒策略改进步骤，旨在无需额外神经网络训练的情况下提升鲁棒性。具体而言，我们利用对抗性 rollout 下的模型预测控制，该控制通过有界不确定性集内的投影梯度下降进行近似。此外，这些离线 rollout 在执行时考虑并缓解了分布外问题。通过在受扰动的 Gymnasium MuJoCo 环境中评估算法，并考虑后验推理场景的计算限制，验证了所提方法在鲁棒性上的显著提升。

## Abstract
To improve the real-world applicability of reinforcement learning (RL), the field of adversarially robust RL studies how to train agents under adversarial environment perturbations. In this setting, a protagonist agent optimizes a policy under environmental perturbations from an adversary, resulting in a zero-sum Markov game. When adversarially robust RL is combined with model-based RL, the adversary can target a learned transition model instead of the training environment. Extending this idea, this work introduces post-hoc robustification of deep RL agents at inference time. By using the learned model in combination with a trained nominal policy, our approach performs a robust policy improvement step. The goal is to improve robustness without any additional training of neural networks. Specifically, we utilize model-predictive control under adversarial rollouts, which are approximated via projected gradient descent within a bounded uncertainty set. Furthermore, these offline rollouts are performed while considering and mitigating out-of-distribution issues. The proposed methodology is validated by demonstrating significant improvements in robustness when the algorithm is evaluated in perturbed Gymnasium MuJoCo environments, while considering the computational limitations of the post-hoc inference setting.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：强化学习（RL）在现实部署中面临环境扰动（如 sim2real 或 sim2sim）导致的性能下降问题。现有的对抗鲁棒 RL 方法大多要求训练阶段引入对抗攻击，这往往带来训练不稳定、性能下降等问题，且需要修改训练过程。
- **研究动机**：探索 **推理阶段（inference time）的鲁棒化**，即在不修改训练过程、不额外训练神经网络的前提下，提升已训练好的基于模型 RL（MBRL）智能体在扰动环境中的鲁棒性。这种“后验鲁棒性”（post-hoc robustness）允许在部署时灵活定义不确定性集和鲁棒性/最优性权衡。
- **整体含义**：该工作提出一种实用的推理阶段鲁棒化框架，利用已学习的转移模型和标称策略（nominal policy），通过模型预测控制（MPC）结合对抗性轨迹展开来优化策略，同时处理分布外（OOD）问题。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- 在推理时，对每个状态使用模型预测控制（MPC）在**对抗性轨迹**上优化局部策略，以抵抗环境扰动。其中对抗性轨迹通过对学到的转移模型的预测状态施加**投影梯度下降（PGD）扰动**来实现，使新状态落入不确定性集内且最小化标称值函数。

### 关键技术细节
1. **鲁棒 MPC 策略**：
   - 定义 rollout 值函数 \( V^{\text{rollout}}_k(s_k) = r(s_k, \pi(s_k)) + \gamma V^{\text{rollout}}_{k+1}(s_{k+1}) \)，其中 \( s_{k+1} \sim \arg\min_{P\in\mathcal{P}} E_{s'\sim P(\cdot|s,a)}[V^\pi(s')] \)。
   - 使用滚动深度 \(N\)，最终步用标称值函数 \(V^\pi(s_N)\)。
   - 外层优化：通过 MPPI 算法近似求解 $\pi^{\text{MPC}}_s = \arg\max_{\pi_s} E_{\pi_s}[V^{\text{rollout}}(s)]$。

2. **对抗性状态扰动（PGD）**：
   - 扰动 $\delta$ 通过归一化 PGD 更新：$\delta \leftarrow \text{Proj}_{\|\delta\|_2 \le r_2} \left( \delta - \alpha \frac{\nabla_{(s'+\delta)}V_\psi(s'+\delta)}{\|\nabla_{(s'+\delta)}V_\psi(s'+\delta)\|_2} \right)$，投影到 L2 球内（半径 $r_2$）。
   - 以此模拟 OTC 不确定性集内的最坏情况转移。

3. **处理分布外（OOD）问题**：
   - 使用学习模型集成（ensemble）的 **方差作为 epistemic uncertainty**，设定阈值 $h$，当 $>h$ 时截断 rollout，避免模型被滥用。
   - 实验表明无此机制时性能甚至低于基线。

## 6. 论文的主要结论与发现

1. **方法有效**：在 Reacher-v4 和 Hopper-v4 的多种单/双参数扰动下，Robust MPC 显著提高鲁棒性，减少扰动导致的性能衰减。
2. **关键组件验证**：
   - OOD 缓解机制至关重要（无该机制时性能低于基线）。
   - 对抗扰动是鲁棒性提升的主导因素（Non-Robust MPC 提升有限，尤其在 Hopper-v4 上几乎无改善）。
3. **超参数影响**：
   - 增加 rollout 深度 N 可显著提升鲁棒性（Hopper-v4 上 N=40 优于 N=1/10/20）。
   - 不确定性集半径 r2 存在稳定区间（0.045~0.075），过大则因模型误差导致性能下降。
4. **计算成本**：推理时单步耗时在高端 GPU 上约 0.1 秒（Hopper-v4），在可用范围内。
