## 摘要
机器人足球是多智能体强化学习的一个具有挑战性的试验平台，因为它结合了部分可观测性、合作与对抗交互、稀疏奖励以及长期战术行为。RoboCup 2D足球仿真（RCSS2D）提供了一个成熟的机器人足球平台，但其面向竞争的服务器-客户端架构难以直接与现代基于Python的MARL工作流程一起使用。我们提出R2D-RL，这是一个强化学习环境，通过共享内存通信和周期级同步将RCSS2D和基于HELIOS的玩家客户端连接到Python MARL接口。R2D-RL支持全场和基于场景的训练，具有可配置的对手、基础离散和混合参数化动作空间、动作掩码、基于预期控球价值（EPV）的奖励塑造以及并行执行。我们提供了前场射门场景和一个11对11的全场基准测试，以及基线结果。

## Abstract
Robot soccer is a challenging testbed for multi-agent reinforcement learning because it combines partial observability, cooperative and adversarial interaction, sparse rewards, and long-horizon tactical behavior. RoboCup 2D Soccer Simulation (RCSS2D) provides a mature robot-soccer platform, but its competition-oriented server-client architecture is difficult to use directly with modern Python-based MARL workflows. We introduce R2D-RL, a reinforcement learning environment that connects RCSS2D and HELIOS-based player clients to a Python MARL interface through shared-memory communication and cycle-level synchronization. R2D-RL supports full-field and scenario-based training with configurable opponents, Base discrete and Hybrid parameterized action spaces, action masks, expected possession value (EPV)-based reward shaping, and parallel execution. We provide front-goal scenarios and an 11-vs-11 full-field benchmark, together with baseline results.

- **步进同步通信协议**：使用共享内存序列计数器（obs_seq, act_seq, done_seq）组织交互。
  - 每个受控玩家在 play_on 模式下发布观测和动作掩码，递增 obs_seq。

  - 玩家等待 act_seq = obs_seq，读取动作并执行，完成后更新 done_seq。
  - 环境等待所有玩家 done_seq = act_seq，确认动作已消费。
- **仅暴露 play_on 模式**：非 play_on 模式（如开球、任意球）由 RCSS2D 规则和 HELIOS 回退行为处理，不进行策略决策。

## 6. 论文的主要结论与发现

1. **环境可用性**：R2D-RL 能够将 RCSS2D 转化为步进同步的 MARL 环境，支持并行采样，Base 和 Hybrid 动作空间均可训练。
2. **动作掩码的重要性**：在 QMIX 中，面对有压力的防守（Blocked Shot），动作掩码至关重要（无掩码时目标率为 0.00±0.00，有掩码时达到 0.88±0.02）。MAPPO 对掩码的依赖较小。
3. **MaxEPV 塑造奖励的效果**：在多攻击者场景中，无 MaxEPV 和无掩码的设置几乎无法得分；MaxEPV 有助于提升目标率。
4. **渐进难度梯度**：从前场场景到全场，任务复杂性递增，算法性能下降。Compact Defense 最困难，仅 MAPPO 在 MaxEPV+掩码设置下取得非零目标率（0.32±0.09）。
5. **全场基线**：30M 步后，MAPPO 目标差 -10.61±1.64，QMIX -19.00±8.60，ParaDQN -25.90±2.69。MAPPO 最稳定，QMIX 种子方差大。所有算法与规则对手相比仍处于劣势，显示全场学习潜力巨大。
6. **限制**：当前策略表现出固定方向的无球移动、缺乏明确的禁区附近守门员行为、对死球情况（任意球等）无学习。
