## 摘要
为任意几何体生成高质量网格仍然是计算工程中的一个基本瓶颈，往往需要启发式调整和半手动工作流程。在本文中，我们介绍了Dmsh，这是首个完全自动化的强化学习管道，它将几何分解和四边形网格生成统一在一个基于学习的框架内。Dmsh通过三个协调的智能体将问题分解为拓扑简化、几何正则化和网格生成。网格化过程被建模为马尔可夫决策过程，并使用带有解耦评论家的参数化柔性演员-评论家架构求解，从而能够高效探索混合离散-连续动作空间。课程学习策略确保从简单域到高度复杂几何体的可扩展性，抑制种子方差。通过设计，递归分解能够并行对子区域进行网格划分，产生全局兼容的全四边形网格，无需事后修正。在广泛的基准测试中，Dmsh在自动化、鲁棒性和网格质量方面一致优于现有方法，为基于学习的网格生成建立了新范式。

## Abstract
Generating high-quality meshes for arbitrary geometries remains a fundamental bottleneck in computational engineering, often demanding heuristic tuning and semi-manual workflows. In this paper, we introduce Dmsh, a first fully automated reinforcement learning pipeline that unifies geometric decomposition and quadrilateral mesh generation within a single learning-based framework. Dmsh decomposes the problem through three coordinated agents handling topology simplification, geometric regularization, and mesh generation. The meshing process is formulated as a Markov Decision Process and solved using a parametric Soft Actor-Critic architecture with decoupled critics, enabling efficient exploration of a hybrid discrete-continuous action space. A curriculum learning strategy ensures scalability from simple domains to highly complex geometries, suppressing seed variance. By design, the recursive decomposition enables parallel meshing of subregions, yielding globally conforming all-quadrilateral meshes without post hoc correction. Across a wide range of benchmarks, Dmsh consistently outperforms existing methods in automation, robustness, and mesh quality, establishing a new paradigm for learning-based mesh generation.

## 2. 论文提出的方法论

### 核心思想
- 将网格生成过程建模为**马尔可夫决策过程（MDP）**，采用**多智能体强化学习**框架 **Dmsh**，包含三个协同工作的智能体：
  1. **网格生成智能体（Meshing Agent）**：基于推进前沿法依次放置四边形单元，使用参数化 Soft Actor-Critic（P-SAC）处理混合离散‑连续动作空间。
  2. **块分解智能体（Block Decomposition Agent）**：基于视觉输入（局部+全局图像与向量状态），递归地将带强凹角的域分割为简单子区域。
  3. **孔分解智能体（Hole Decomposition Agent）**：纯视觉智能体，将包含内部孔洞的几何体分割为多个简单连通区域。
- 所有智能体采用改进的 **Soft Actor-Critic（SAC）** 算法，结合课程学习、解耦评论家、双温度调节等技术，确保有效探索和稳定收敛。

### 关键技术细节
- **网格生成智能体**：
  - **状态表示**：局部旋转不变极坐标系，包含左右各 3 个邻接顶点、3 条扇形射线、以及全局面积比例。
  - **动作空间**：混合型：类型 0（缝合，离散）和类型 1（创建新节点，连续参数 \([-1,1]^2\)）。
  - **P-SAC 架构**：统一演员网络分裂成离散头和连续头；评论家按动作类型解耦为 4 个网络（每类 2 对 twin Q-network），使用掩码损失更新；采用两个独立的温度参数 \(\alpha_{\text{disc}}, \alpha_{\text{cont}}\) 自动调节探索。
  - **奖励设计**：基于缩放雅可比（Scaled Jacobian）的单元质量、边界平滑性、窄带惩罚、顶点价惩罚等复合奖励。
  - **失败恢复**：包含几何扩张与约束拉普拉斯平滑，防止拓扑死锁。
- **块分解智能体**：
  - 基于双通道图像（局部 128×128 + 全局 128×128）与向量状态融合，伪孪生 CNN 编码，SAC 预测单条切割线的角度。
  - 强制奇偶校验：若切分后子区域顶点数为奇数，则寻找初始边界上的最长边并二分。
  - 奖励：切口正交性、子域平衡、移除凹角、平滑性。
- **孔分解智能体**：
  - 纯视觉输入（局部+全局二值掩码），智能体位于孔洞的“不可达极点”（最大内切圆中心），预测两个切割角度，执行双射线切割。
  - 奇偶校验同块分解，优先分割初始边界上的长边。
  - 奖励：切割有效性（避免窄条）、凸包比、角度分数、边长比。
- **课程学习**：第一阶段在简单正方形上预训练 100k 步；第二阶段在多种几何体上集成训练 1.5M–2M 步，采用软最大选择策略降低种子方差。

### 算法流程（文字描述）
1. 输入几何体 → 若含孔洞，孔分解智能体递归切割，生成无孔子域。
2. 每个无孔子域 → 块分解智能体递归切分，消除强凹角，生成拓扑简单的子块。
3. 每个子块 → 网格生成智能体独立并行运行，基于 MDP 逐步放置四边形，直到边界点数 ≤ 4。
4. 所有子块网格全局拼接，保证节点一致性，得到完整全四边形网格。
