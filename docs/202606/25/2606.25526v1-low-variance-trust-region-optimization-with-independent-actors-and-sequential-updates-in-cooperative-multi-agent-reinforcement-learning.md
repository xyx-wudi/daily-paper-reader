## 摘要
合作多智能体强化学习假设每个智能体共享相同的奖励函数，并可以利用单智能体的信任区域框架进行有效训练。独立演员设置不依赖其他智能体的动作，而是考虑每个智能体仅基于其局部信息行动，因此具有更灵活的应用。然而，在顺序更新框架中，需要在每个智能体策略步骤后重新估计联合优势函数。尽管重要性采样在实践中取得了成功，但更新后的优势函数存在指数级高方差问题，可能导致不稳定的收敛。在本工作中，我们首先通过经验和理论分析了高方差优势。为克服这一局限，我们引入了一个裁剪目标来控制顺序更新中优势波动的上界。通过所提出的目标，我们提供了一个单调界，并亚线性收敛到ε-纳什均衡。我们进一步利用裁剪目标推导了两种新的实用算法。在三个流行的多智能体强化学习基准上的实验结果表明，我们的方法在大多数环境中优于测试的基线。通过仔细分析不同的训练设置，我们的方法突出了稳定的收敛特性和期望的低优势方差估计。为便于复现，我们的源代码已在https://github.com/giangbang/Low-Variance-Trust-Region-MARL公开。

## Abstract
Cooperative multi-agent reinforcement learning assumes each agent shares the same reward function and can be trained effectively using the Trust Region framework of single-agent. Instead of relying on other agents' actions, the independent actors setting considers each agent to act based only on its local information, thus having more flexible applications. However, in the sequential update framework, it is required to re-estimate the joint advantage function after each individual agent's policy step. Despite the practical success of importance sampling, the updated advantage function suffers from exponentially high variance problems, which likely result in unstable convergence. In this work, we first analyze the high variance advantage both empirically and theoretically. To overcome this limitation, we introduce a clipping objective to control the upper bounds of the advantage fluctuation in sequential updates. With the proposed objective, we provide a monotonic bound with sub-linear convergence to $ε$-Nash Equilibria. We further derive two new practical algorithms using our clipping objective. The experiment results on three popular multi-agent reinforcement learning benchmarks show that our proposed method outperforms the tested baselines in most environments. By carefully analyzing different training settings, our proposed method is highlighted with both stable convergence properties and the desired low advantage variance estimation. For reproducibility purposes, our source code is publicly available at https://github.com/giangbang/Low-Variance-Trust-Region-MARL.

- **问题**：在完全合作的多智能体强化学习（MARL）中，**独立演员（independent actors）** 与**顺序更新（sequential updates）** 相结合的信任区域方法（如HAPPO/HATRPO）具有理论上的单调改进保证。然而，由于每个智能体策略更新时需通过重要性采样重新估计联合优势函数，且重要性权重是各智能体比值的乘积，导致**优势估计方差随智能体数量指数增长**，引致训练不稳定、收敛困难。
- **意义**：现有的信任区域MARL工作（如HAPPO）虽在实践中成功，但忽视了高方差问题。本文系统分析了该问题的理论和经验表现，并提出一种**低方差信任区域优化框架**，在保持理论保证的同时提升稳定性和性能。

## 2. 论文提出的方法论

### 核心思想
- 对重要性采样权重 \(\mathfrak{R}^{1:m-1}_{\tilde{\pi},\pi}\) 进行**裁剪（clipping）**，限制其上限为1，从而控制优势波动的上界，降低方差。
- 定义新的代理目标函数：
  \[
  L^{1:m,\text{clip}}_\pi(\tilde{\pi}^{1:m-1}, \hat{\pi}^m) = \mathbb{E}_{s\sim d^\pi_\rho, a\sim\pi}\left[ \left(\frac{\hat{\pi}^m(a^m|s)}{\pi^m(a^m|s)} - 1\right) \min\left(\mathfrak{R}^{1:m-1}_{\tilde{\pi},\pi}(s,a), 1\right) A^\pi(s,a) \right]
  \]
- 在顺序更新中，每个智能体 \(m\) 优化该裁剪目标加上反向KL散度惩罚（\(\beta_k D_{\text{KL}}(\pi^m \| \pi^m_k)\)），算法如 Algorithm 1 所示：每轮随机排列智能体，依次更新策略。

### 关键理论结果
- **单调改进界**（Lemma 3）：新目标下界了真实性能差，损失项仅与智能体策略间的L1距离有关。
- **收敛性**（Theorem 4）：算法以 \(O(1/\sqrt{K})\) 亚线性收敛到 \(\varepsilon\)-纳什均衡，速率与智能体数 \(N\)、动作空间大小 \(A\) 等参数相关。
- 方差上界被证明为常数 \( \epsilon^2 \)，不再随智能体数增长。

### 算法流程（文字描述）
1. 初始化均匀随机联合策略 \(\pi_0\)。
2. 对每轮 \(k\)：
   - 计算优势函数 \(A^{\pi_k}(s,a)\) 并选择步长 \(\beta_k\)。
   - 随机排列所有智能体。
   - 对每个智能体 \(m=1,\dots,n\) 依次执行：
     \[
     \pi^{m}_{k+1} \leftarrow \arg\max_{\pi^m} \left[ L^{1:m,\text{clip}}_{\pi_k}(\pi^{1:m-1}_{k+1}, \pi^m) - \beta_k \mathbb{E}_{\rho^{\pi_k}} D_{\text{KL}}(\pi^m \| \pi^m_k) \right]
     \]
3. 输出均匀采样的策略 \(\bar{\pi} = \pi_k\)。

- **高方差问题确实存在**：理论和实验均证明顺序更新中优势估计方差可随智能体数指数增长（图1、图5）。
