## 摘要
联邦强化学习使分散的智能体能够在不交换原始轨迹的情况下协同改进策略或价值估计。然而，FedAvg风格的参数平均在函数空间上并不一致：当客户端使用异构编码器甚至相同的非线性网络时，平均参数并不对应于任何公共函数空间中客户端价值函数的加权平均。我们提出FedQHD，一种使用超维（随机特征）状态编码器和线性读出层的联邦Q学习方法，使得Q函数在状态上非线性但在可训练参数上线性。这种线性结构实现了闭式聚合。对于共享编码器，函数空间共识更新与局部读出矩阵的加权平均完全一致。对于异构编码器，服务器通过在共享锚点状态集上平均客户端Q值来构建全局教师，每个客户端通过一次岭回归投影将该教师编译为其局部表示。我们将联邦间隙——将联邦教师编译为异构客户端表示时产生的误差——形式化为相对于客户端特定投影的误差。我们证明该间隙可分解为子空间错位、锚点集条件和正则化偏差。我们进一步确定锚点与维度之比 $m \geq D_i$ 作为良好条件区域，在该区域内间隙简化为编码器异构性基底的倍数。在四个连续状态、离散动作控制基准测试中，FedQHD匹配或优于FedAvg风格基线和基于蒸馏的替代方法，同时所需计算量大幅减少，并且联邦间隙对编码器维度的经验依赖性符合我们的理论分析。

## Abstract
Federated reinforcement learning enables decentralized agents to collaboratively improve policies or value estimates without exchanging raw trajectories. However, FedAvg-style parameter averaging is not function-space consistent: when clients use heterogeneous encoders or even identical nonlinear networks, averaged parameters need not correspond to the weighted average of client value functions in any common function space. We propose FedQHD, a federated Q-learning method using hyperdimensional (random-feature) state encoders with a linear readout, so that Q-functions are nonlinear in state yet linear in trainable parameters. This linear structure enables closed-form aggregation. With a shared encoder, the function-space consensus update coincides exactly with weighted averaging of local readout matrices. With heterogeneous encoders, the server constructs a global teacher by averaging client Q-values on a shared anchor-state set, and each client compiles this teacher into its local representation via a single ridge projection. We formalize the federation gap -- the error incurred when compiling a federated teacher into a heterogeneous client representation -- relative to a client-specific oracle projection. We show that this gap decomposes into subspace misalignment, anchor-set conditioning, and regularization bias. We further identify the anchor-to-dimension ratio $m \geq D_i$ as the well-conditioned regime in which the gap reduces to a multiple of the encoder heterogeneity floor. On four continuous-state, discrete-action control benchmarks, FedQHD matches or outperforms FedAvg-style baselines and distillation-based alternatives while requiring substantially less computation, and the empirical dependence of the federation gap on encoder dimension matches our theoretical analysis.

## 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：联邦强化学习（FedRL）中，经典的 FedAvg 参数平均方法在函数空间上不一致。当客户端使用异构编码器（不同维度或不同随机特征映射）甚至相同结构的非线性网络时，参数加权平均并不对应任何公共函数空间中真实价值函数的加权平均，导致策略更新偏差。
- **背景与动机**：实际联邦部署中设备常具有异构架构（不同编码器、特征维度），同时数据无法中心化。现有方法如知识蒸馏虽能处理异构性，但需要迭代的师生优化，对超参数敏感且不稳定。作者希望找到一种**闭式聚合**方案，既能保证函数空间一致性，又能处理异构表示，且避免迭代优化。

## 2. 方法论：核心思想、关键技术细节与算法流程
### 核心思想
- 使用**超维计算（HDC，即随机傅里叶特征）** 作为状态编码器，配合线性读出层，使得 Q 函数在状态上非线性但在可训练参数上线性。
- 利用线性结构的特性，将联邦聚合转化为参数空间的加权平均（同构编码器）或锚点集上的岭回归投影（异构编码器）。

### 关键技术细节
- **QHD 模型**：每个客户端 i 使用编码器 Φ_i 将状态映射为 D_i 维超向量，对每个动作 a 维护参数 w_{i,a}，Q_i(s,a) = ℜ(⟨Φ_i(s), w_{i,a}⟩)。
- **同构编码器**：若所有客户端共享相同 Φ，则函数空间共识等价于参数加权平均：W_glob = Σ π_i W_i。无需数据交换。
- **异构编码器**：
  - 服务器采样一组共享锚点状态 S_ref（如 m=200），各客户端评估其 Q 值并上传给服务器。
  - 服务器计算加权平均教师 Q_ref_glob = Σ π_i Q_ref_i。
  - 每个客户端通过**闭式岭回归**将教师编译到自身特征空间：W_glob_i = (X_i^H X_i + λI)^{-1} X_i^H Q_ref_glob。等价于 W_glob_i = X_i^H (G_i + λI)^{-1} Q_ref_glob（利用 Woodbury 恒等式，求逆大小为 m×m 而非 D_i×D_i）。
- **理论分析**：给出联邦间隙 ∆_i = 客户端 Oracle 投影 - 编译后 Q 函数的点式界，分解为 **(I) 编码器异质性**（子空间错位）、**(II) 锚点放大**（取决于锚点 Gram 矩阵的最小正特征值 γ_i）、**(III) 脊收缩偏差**（由 λ 引起）。证明当 m ≥ D_i 时，(II) 退化为常数倍异质性基底。

### 算法流程（文字描述）
1. **初始化**：共享或各客户端独立采样编码器 Φ_i，参数 W_i=0，服务器生成锚点集 S_ref。
2. 每轮联邦迭代：
   - 各客户端本地执行 K 个 episode 的 QHD 半梯度 TD 更新（使用目标网络）。
   - 同构：客户端上传本地 W_i；服务器计算加权平均 W_glob，广播。
   - 异构：客户端上传锚点 Q 值 Q_ref_i；服务器计算 Q_ref_glob；每个客户端通过一次岭回归更新自身 W_glob_i（无需广播全局参数，仅编译教师）。
3. 重复直至收敛。

## 8. 不足与局限
1. **实验环境局限**：仅测试连续状态、离散动作的简单控制任务，未验证高维图像状态（如 Atari）或连续动作（需 actor-critic）。对实际复杂应用（自动驾驶、机器人）的推广性不明。
2. **编码器固定而非学习**：随机傅里叶特征编码器非学习生成，可能不如自学习特征表达能力强。论文也提到扩展到“学习编码器”是未来工作。
3. **理论假设较强**：假设真实 Q^* 属于公共 RKHS 且 RKHS 范数有界，且 ν_ref ≪ ν_i。实际中可能不严格满足，尤其在环境异构时。
4. **锚点集依赖**：锚点需要覆盖所有客户端状态支持，若客户端状态分布差异大，需大量锚点，可能增加通信成本。
