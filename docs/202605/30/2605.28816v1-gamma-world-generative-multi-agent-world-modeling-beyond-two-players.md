## 摘要
用于交互式视频生成的世界模型主要关注单智能体设置，即从单一控制信号生成未来观测。然而，许多生成环境需要多智能体交互：多个玩家、机器人或具身智能体在共享空间中同时行动。将世界模型扩展到此类设置需要原则性的多智能体设计：智能体应保持独立可控、置换对称，并支持高效推理，同时保持时间和视角的一致性。本文提出了一种用于交互式模拟的生成式多智能体世界模型。它引入了Simplex Rotary Agent Encoding，这是3D RoPE的一种无参数扩展，将智能体表示为旋转角度空间中正则单纯形的顶点。这赋予每个智能体不同的相位，同时使所有智能体置换等价，从而实现可扩展的智能体身份，而无需学习每个槽位的身份或固定的智能体顺序。为避免智能体间的密集全连接注意力，我们进一步提出了稀疏中心注意力（Sparse Hub Attention），其中可学习的中心令牌（hub tokens）协调智能体间的令牌交互，将跨智能体注意力成本从智能体数量的二次方降至线性。为了实现实时推演，我们将全上下文扩散教师模型蒸馏为因果学生模型，该模型通过KV缓存顺序生成时间块，从而实现24 FPS的动作响应生成。在多玩家虚拟环境中的实验表明，与基于槽和密集注意力的基线相比，我们的模型在视频保真度、动作可控性和智能体间一致性方面均有所提升，并且无需额外训练即可从两个玩家泛化到四个玩家。

## Abstract
World models for interactive video generation have largely focused on single-agent settings, where future observations are generated from a single control signal. However, many generated environments require multi-agent interaction: multiple players, robots, or embodied agents act simultaneously within a shared space. Scaling world models to such settings requires a principled multi-agent design: agents should remain independently controllable, permutation-symmetric, and support efficient inference while maintaining consistency across time and perspectives. In this paper, we present our generative multi-agent world model for interactive simulation. It introduces Simplex Rotary Agent Encoding, a parameter-free extension of 3D RoPE that represents agents as vertices of a regular simplex in rotary angle space. This gives each agent a distinct phase while making all agents permutation-equivalent, enabling scalable agent identity without learned per-slot identities or a fixed agent ordering. To avoid dense all-to-all attention across agents, we further propose Sparse Hub Attention, where learnable hub tokens mediate token interaction across agents, reducing cross-agent attention cost from quadratic to linear in the number of agents. For real-time rollout, we distill a full-context diffusion teacher into a causal student that generates temporal blocks sequentially with KV caching, enabling action-responsive generation at 24 FPS. Experiments in multiplayer virtual environments show that our model improves video fidelity, action controllability, and inter-agent consistency over slot-based and dense-attention baselines, while generalizing from two to four players without additional training.

  - 随智能体数量增加计算成本呈平方增长；
  - 学习的身份嵌入破坏了置换对称性，无法零样本泛化到更多智能体。
- **整体目标**：提出一种可扩展、置换对称、高效的生成式多智能体世界模型，支持**双玩家以上**的实时交互模拟。

## 2. 论文提出的方法论：核心思想、关键技术细节
### 2.1 整体框架
- 基于DiT（Diffusion Transformer）架构，采用流匹配（flow-matching）目标。
- 输入为P个智能体的同步历史观测和动作序列，输出每个智能体的下一帧观测。
- 采用两阶段训练：先训练一个**双向扩散教师**（能访问完整上下文），再将其知识蒸馏到**因果学生模型**中（支持流式生成和KV缓存）。

### 2.2 Simplex Rotary Agent Encoding（单纯形旋转智能体编码）
- **核心思想**：用几何方式赋予每个智能体独特的相位，同时确保所有智能体两两之间的旋转距离相等，从而满足置换对称性。
- **技术细节**：
  - 将3D RoPE扩展为4D：增加智能体轴(p)，头维度分配为(d_t, d_p, d_h, d_w)。
  - 在d_p/2维的智能体角度空间中，构造正整数V（池大小）个单纯形顶点：s_v = √(V/(V-1)) Q (e_v - (1/V)·1)，其中Q是线性等距映射。顶点具有单位范数且两两距离恒定为sqrt(2V/(V-1))。
  - 每个智能体被分配一个不同的顶点，相位角θ_p = α·s_{π(p)}，α控制分离强度。
  - 该编码无需参数、无需学习，并支持零样本增加智能体数量（只需从池中选取未使用顶点）。
- **公式**：R_4D(t,p,h,w) = diag(R_t(t), R_simp(π(p)), R_h(h), R_w(w))

### 2.3 Sparse Hub Attention（稀疏中心注意力）
- **核心思想**：避免智能体间密集的全连接注意力（O(P²)），改为引入少量可学习的**中心令牌（hub tokens）**作为共享通信状态。
- **通信拓扑**：
  - 每个智能体令牌只与自己流内的令牌和所有hub tokens交互（自注意力）。
  - Hub tokens与所有智能体令牌以及其他hub tokens交互。
  - 直接跨智能体流之间的注意力被屏蔽，信息通过“智能体→hub→智能体”两跳路径传递。
- **复杂度**：从O(P² n² L²)降低到O(P n L (n L + n K) + n K (P n L + n K))，即对智能体数量P线性增长。
- **实现**：每帧设置K个hub tokens（实验取K=8），它们从可学习矩阵H中广播，并屏蔽输出，只作为内部通信载体。

### 2.4 训练与推理流程
- **阶段1**：训练双向教师模型（全注意力、共享噪声水平）。
- **阶段2**：训练因果学生模型（结合块因果注意力与Sparse Hub注意力，采用Diffusion Forcing中每块独立噪声水平）。
- **阶段3**：利用Self-Forcing + DMD（分布匹配蒸馏）将多步因果学生蒸馏为**4步**生成器。
- **推理**：使用KV缓存逐块生成，达到24 FPS实时推演。每个智能体维护独立KV缓存，hub tokens维护共享缓存。
