## 摘要
标准的实用拜占庭容错（PBFT）协议专为稳定的有线环境设计，在移动自组织网络等部署场景中展现出严重漏洞，使其易受女巫攻击、拜占庭共谋和消息操纵等复杂威胁的影响。现有的静态防御机制不足以应对这些攻击的智能化和协同性。为应对这一挑战，本文利用Agentic AI范式构建了一个分布式多智能体系统，其中每个共识节点配备一个智能体。这些智能体采用层次化多智能体深度Q网络（MADQN）算法，实时学习和执行主动安全策略。通过观察本地网络行为、消息一致性以及动态维护的声誉评分，智能体协同识别可疑行为，并在标准PBFT法定人数和成员规则下推荐防御措施，从而提升共识过程的完整性。我们将所得框架称为Agentic-SecPBFT。随后，我们对关键攻击向量进行形式化建模并开展大量仿真。结果表明，Agentic-SecPBFT实现了95.0%的攻击检测率，误报率为1.8%。与主流PBFT变体相比，在33%恶意节点下，其吞吐量平均提高3.1倍，延迟降低56%，为去中心化无线系统提供了鲁棒且自适应的安全解决方案。

## Abstract
The standard Practical Byzantine Fault Tolerance (PBFT) protocol, designed for stable, wired environments, exhibits critical vulnerabilities when deployed in settings like mobile ad-hoc networks, thus making it susceptible to sophisticated threats such as Sybil attacks, Byzantine collusion, and message manipulation. Existing static defense mechanisms are ill-equipped to handle the intelligent and coordinated nature of these attacks. To address this challenge, this paper leverages the Agentic AI paradigm to build a distributed multi-agent system in which each consensus node is equipped with an intelligent agent. These agents employ a hierarchical Multi-Agent Deep Q-Network (MADQN) algorithm to learn and execute proactive security policies in real-time. By observing local network behavior, message consistency, and dynamically maintained reputation scores, the agents collaboratively identify suspicious behavior and recommend defensive actions under standard PBFT quorum and membership rules, thereby improving the integrity of the consensus process. We refer to the resulting framework as Agentic-SecPBFT. Then, we formally model key attack vectors and conduct extensive simulations. The results demonstrate that Agentic-SecPBFT reaches a 95.0% attack detection rate with a 1.8% false positive rate. Compared with mainstream PBFT variants, it achieves 3.1* higher throughput with 56% lower latency on average under 33% malicious nodes, offering a robust and adaptive security solution for decentralized wireless systems.

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：利用**智能体AI（Agentic AI）** 范式，将每个共识节点转变为自主防御智能体，构建分层多智能体系统。采用**集中训练、分散执行（CTDE）** 范式，结合层次化多智能体深度Q网络（Hierarchical MADQN）实现主动安全策略学习与执行。
- **关键技术细节**：
  - **分层智能体架构**：
    - **本地共识智能体（LCA）**：运行在每个物理节点上，实时感知局部状态（SINR、CSI、协议阶段、投票熵、声誉等），执行离散安全动作（正常参与、标记可疑节点、请求挑战-响应、投票视图变更、临时拒收消息、调整发射功率、预留优先级时隙等）。使用LSTM处理部分可观测性，Dueling DQN结构输出Q值。
    - **集群优化智能体（COA）**：运行在高可用超级节点或SMPC委员会，聚合全局遥测数据，进行全局MADQN训练，更新声誉向量并广播最新策略参数。COA不参与PBFT提交关键路径，其输出仅作为学习指导和建议，无法改变法定人数或永久移除成员。
  - **智能体交互协议（AIP）**：LCA在每轮共识后存储经验，定期（如每100区块）上传经验至COA；COA聚合后更新全局Q网络和声誉矩阵，并通过签名、版本化的AIP_POLICY_UPDATE广播至LCA。
  - **动态信任与声誉管理**：引入信道感知可靠度 \(P_{\text{success}}(i,j) = 1 - P_{\text{out}}(i,j)\)（基于SINR阈值 \( \gamma_{\text{th}} \) 的链路中断概率），定义物理确定性因子 \(\Psi_{\text{phy}} = P_{\text{success}}(1 - \xi_v)\)（\(\xi_v\) 为移动速度惩罚）。声誉更新使用投影法保持[0,1]范围：  
    \[
    R_{t+1}^{i,j} = \Pi_{[0,1]}\left(R_t^{i,j} + (1-\mu)[I_{\text{honest}}\omega_{\text{reward}} - I_{\text{suspect}}\omega_{\text{penalty}}\Psi_{\text{phy}}]\right)
    \]
    从而在信道差时降低惩罚，好信道时加大惩罚。
  - **层次化MADQN算法**：
    - **状态空间**：包含共识特征（阶段、视图、历史投票、投票熵）、网络特征（SINR向量、CSI空间相关、到达时间方差、平均干扰强度、并发发送数）、声誉特征（本地声誉、声誉方差）。
    - **动作空间**：7个离散动作（参与正常、标记可疑、降低声誉、请求挑战、投票视图变更、临时拒收、调整功率、预留时隙）。
    - **转移概率**：分解为信道状态转移 × 协议状态转移 × 声誉状态转移，显式耦合信道与协议动态。
    - **奖励函数**：多目标加权，包括共识成功奖励、安全检测/误报惩罚、活性奖励（避免不必要视图变更、维持SINR）。
  - **算法流程**：
    - **集中训练（COA）**：初始化全局Q网络和目标Q网络，聚合所有LCA的经验，采样批次更新全局网络，软更新目标网络，广播更新。
    - **分散执行（LCA）**：每时间步感知状态→LSTM聚合→ϵ-贪婪选择动作→执行防御（受PBFT安全护栏约束）→更新本地声誉和奖励→存储经验→定期与COA同步。

### 8. 不足与局限

- **应用限制**：
  - 框架需要节点具备CSI/SINR感知能力，对无此功能的IoT设备不适用。
  - 训练阶段算力要求较高（多GPU），可能限制资源受限网络的部署。
  - 动作空间包括“调整发射功率”等，可能受无线电管制法规约束。
