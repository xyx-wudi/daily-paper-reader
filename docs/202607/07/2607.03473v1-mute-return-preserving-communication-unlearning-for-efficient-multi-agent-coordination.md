## 摘要
智能体间通信对于在部分可观测条件下协调多智能体强化学习（MARL）智能体以在合作游戏中有效执行至关重要；然而，现实世界的带宽限制要求稀疏交互。先前的方法主要通过优化信息论代理来解决这一权衡。我们认为这些统计代理从根本上与真实目标不一致：一条消息可能信息量很大，但与任务的联合回报无关。在这项工作中，我们提出了面向目标效率的消息遗忘（MUTE）框架，该框架将通信减少视为一个价值引导的机器遗忘问题。MUTE使用基于注意力的估计器严格量化反事实消息价值，并从无任何通信约束训练的策略中系统地遗忘低价值消息的传输。这是通过一个双目标机制实现的，该机制在保持原始联合策略回报的同时强制通信稀疏性。我们推导了由这种稀疏化引起的性能差距的理论上界，保证了可控的回报退化。我们还在各种复杂的多智能体环境中对MUTE进行了实证评估，在保持与最先进基线相当性能的同时实现了80%至90%的带宽减少。

## Abstract
Inter-agent communication is critical for coordinating Multi-Agent Reinforcement Learning (MARL) agents under partial observability to perform effectively in cooperative games; however, real-world bandwidth constraints demand sparse interactions. Prior approaches primarily address this trade-off by optimizing information-theoretic surrogates. We argue that these statistical proxies are fundamentally misaligned with the true objective: a message can be highly informative yet irrelevant to the joint return of the task. In this work, we propose Message Unlearning for Targeted Efficiency (MUTE), a framework that views communication reduction as a value-guided machine unlearning problem. MUTE rigorously quantifies the Counterfactual Message Value using an attention-based estimator, and systematically unlearns the transmission of low-value messages from a policy trained without any communication constraints. This is achieved through a dual-objective mechanism that enforces communication sparsity while preserving the return of the original joint policy. We derive a theoretical upper bound on the performance gap induced by this sparsification, guaranteeing controlled return degradation. We also empirically evaluate MUTE on various complex multi-agent environments, achieving 80% to 90% bandwidth reduction while maintaining performance comparable to state-of-the-art baselines.

- **核心问题**: 在部分可观测的多智能体强化学习（MARL）协调任务中，智能体间的通信是关键，但现实世界的带宽限制要求通信必须稀疏。现有的通信减少方法（如NDQ、IMAC）主要依赖优化信息论代理（如互信息）来压缩通信，但作者指出这些**统计代理与真正的任务目标——最大化联合回报（Return）是脱节的**。一条信息量大的消息可能对最终的任务回报无关紧要，甚至是有害的。
- **核心含义**: 本文提出了一种全新的视角，将通信减少问题重新定义为**价值引导的机器遗忘（Unlearning）问题**。目标是训练一个在不损失任务回报的前提下，能尽可能少通信的策略。

#### **2. 方法论**

- **核心思想**: 采用“先学后忘”的策略。首先，训练一个具有**无限制通信能力**的高性能“专家”策略。然后，通过分析和识别哪些消息对维持专家策略的回报是**冗余的**，并系统性地“遗忘”这些低价值消息的生成，最终得到一个稀疏通信策略。
- **关键技术细节**:
    1.  **反事实消息价值 (Counterfactual Message Value, CMV)**: 定义了消息 `m_i` 的价值，通过比较一条消息被发送和未被发送两种反事实场景下的联合动作价值函数 `Q_tot` 的差值来衡量。如果差值接近零，说明该消息是冗余的。
    2.  **消息价值估计器 (Message Value Estimator, MVE)**: 为了高效计算CMV，设计了一个基于**自注意力机制**的神经网络 `V_φ(m)`。它一次性接收所有智能体的消息，并同时输出所有消息的CMV估计值。自注意力机制保证了评估是**排列不变**和**上下文感知**的（例如，如果多条消息包含相同信息，则每条的边际价值都会降低）。
    3.  **通信遗忘 (Communication Unlearning)**: 这是MUTE的核心训练阶段，通过一个**双目标损失函数**进行优化：
        - **稀疏损失 (Sparsity Loss, `L_sparse`)**: 对MVE估计值低于自适应阈值（`λ`）的低价值消息的L1范数进行惩罚，持续施加梯度压力来抑制其信号强度。
        - **锚定损失 (Anchor Loss, `L_anchor`)**: 通过最小化当前策略的个体Q值与冻结的预训练专家策略的个体Q值之间的差距，来**稳定策略**，防止在抑制通信时策略发生灾难性崩溃。
    4.  **理论保证**: 论文推导了**性能退化上界（Theorem 4.2）**，从理论上证明了在MVE估计误差和有界动作值近似误差下，由遗忘过程导致的性能损失是可控制的。

#### **8. 不足与局限**

- **CMV的固有局限**: MUTE使用的CMV是一种**局部、单消息**的边际贡献估计，无法准确捕捉多消息之间的**高阶交互作用**。虽然“锚定损失”在一定程度上缓解了此问题，但理论上仍然存在因同时抑制多条交互消息而导致性能下降的风险。

- **预训练依赖**: MUTE需要一个性能良好的无限制通信策略作为“专家”。在某些任务中，训练这样一个专家策略本身就可能困难或代价高昂。
- **计算成本考量**: 尽管训练时间在有竞争力的范围内，但MUTE的三阶段训练流程（尤其是MVE训练）仍然引入了额外的计算复杂性。在处理超大规模智能体（如>20个）时，其计算开销会显著增长（附录K）。
