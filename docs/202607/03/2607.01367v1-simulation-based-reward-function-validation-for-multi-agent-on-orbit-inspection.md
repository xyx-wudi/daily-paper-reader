## 摘要
针对检测航天器集群控制，一种提出的方法为多智能体强化学习（MARL）。尽管之前的工作已将该方法用于此目的，但所使用的奖励函数侧重于在目标周围达到一组有限的预定检测点。在本工作中，我们基于对轨道上被检测物体的三维重建分析，研究并开发了一种用于MARL检测任务的泛化奖励函数。由于奖励函数是泛化的，可以评估任意位置的任意数量的图像，我们也允许训练后的智能体完全控制图像采集的时机。通过这种方法，我们不仅获得了针对特定MARL检测任务的最佳实践见解，还获得了对MARL背景之外更广泛检测任务具有参考意义的关键要点。

## Abstract
A proposed method for the control of groups of inspection spacecraft is Multi-Agent Reinforcement Learning (MARL). While MARL has already been employed for this purpose in previous work, the reward functions used focus on reaching a finite set of predetermined inspection points around the target. In this work, we study and develop a generalized reward function for the MARL inspection task informed by the analysis of 3D reconstructions of inspected objects in orbit. Because the reward function is generalized such that any number of images at arbitrary locations may evaluated, we also allow trained agents to have complete control over when images are collected. With this approach, we gather insights into best practices for not only the specific MARL inspection task, but also gain key takeaways informative to the broader inspection task outside of a MARL context.

# 论文总结：基于仿真的多智能体在轨检测奖励函数验证

## 1. 核心问题与整体含义（研究动机和背景）
*   **问题**：现有基于多智能体强化学习（MARL）的在轨检测任务中，奖励函数（Reward Function）通常针对一组预先确定的有限检测点进行设计，缺乏灵活性和泛化能力。这限制了智能体自主决策和适应复杂光照、几何条件的能力。
*   **背景**：空间碎片增多，在轨检测是主动移除碎片的关键步骤。传统手动规划检测轨迹效率低，且难以扩展至多航天器协作任务。MARL提供了自动化控制的潜力，但现有奖励函数设计不够灵活。
*   **意义**：本文旨在开发一种**泛化的奖励函数**，允许智能体在任意位置、任意数量、任意时刻采集图像，并通过高保真仿真中生成的3D重建来验证奖励函数的有效性。这不仅能提升MARL检测性能，还能为更广泛的在轨检测任务（不限于MARL）提供通用经验。

## 2. 方法论：核心思想、关键技术细节
*   **核心思想**：基于对目标3D重建质量的分析，设计并验证一种支持任意图像位置和数量的泛化奖励函数。将奖励函数参数化，以便预先权衡重建质量与数据采集成本（燃料、时间等）。
*   **关键技术细节**：
    *   **分层控制架构**：每个航天器配备两个独立智能体：
        *   **上层智能体**：负责高维决策，选择目标点、触发拍照、结束使命。
        *   **下层智能体**：负责低层控制，生成推力命令以到达目标点，同时最小化燃料使用和避免碰撞。
    *   **状态空间**：
        *   下层：自身及所有其他航天器的位置、速度，上层给定的目标点位置。
        *   上层：所有航天器状态、自身单位方向向量、太阳方向、地球阴影标记、已拍照片总数、最近3张图片的位置、最近n张照片的位置（n=100）。
    *   **动作空间**：
        *   下层：三轴推力（连续）。
        *   上层：目标点位置（连续）、拍照/结束信号（离散，通过阈值触发）。
    *   **奖励函数**：核心是**图像价值**，由四个评分因子相乘得到：
        *   **距离评分**：基于拍摄距离与目标距离的差值（指数衰减）。
        *   **分离度评分**：鼓励从不同角度拍摄，惩罚过于相似的图像（基于角度差异）。
        *   **光照评分**：奖励拍摄向阳面，但避免正对太阳（防止自阴影）。
        *   **自阴影评分**：惩罚航天器自身遮挡。
    *   每个智能体的图像奖励是**团队图像价值变化**和**个体图像价值变化**的加权混合（权重0.5）。此外包含燃料消耗、时间、碰撞、超出操作区域等惩罚项。
    *   **课程学习**：由于训练困难，采用课程学习策略。初期几乎消除燃料惩罚并给予平缓奖励以维持行为，然后逐步增加燃料惩罚至最终值。
    *   **动力学模型**：使用Hill-Clohessy-Wiltshire (HCW) 相对动力学模型，假设目标圆形轨道，航天器质量100kg，最大推力10N。
*   **算法**：使用近端策略优化（PPO）算法，采用分散训练-分散执行（DTDE）模式。
