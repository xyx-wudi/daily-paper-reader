## 摘要
基于大语言模型的智能体推荐系统通过自然语言推理在用户偏好建模方面展现出潜力，但受限于以文本为中心的输入和粗粒度的记忆更新，导致智能体容易遗漏视觉证据、受到语义噪声影响并难以捕捉偏好漂移。为解决这些局限，我们提出MMEACR——一个面向推荐的多模态记忆增强智能体协作框架。MMEACR引入双通道记忆架构，将可解释的智能体推理与细粒度多模态匹配分离。在推理通道中，协同工作的用户记忆智能体和物品记忆智能体维护持续的多模态记忆，并通过属性引导的强化-反思机制进行更新。在匹配通道中，从原始交互叙述和物品图像构建解耦的多模态嵌入记忆，以保留结构化记忆更新之外的详细跨模态信号。两个通道通过加权倒数排名融合进行整合，生成稳健且可解释的排名。在三个真实世界领域的实验表明，MMEACR在基于LLM和基于智能体的强基线方法中取得了显著的整体性能，尤其在视觉驱动的推荐场景中表现突出。

## Abstract
Large language model (LLM)-based agentic recommender systems show promise in modeling user preferences through natural-language reasoning, yet they remain limited by text-centric inputs and coarse-grained memory updates, making agents prone to missing visual evidence, semantic noise, and preference drift. To address these limitations, we propose MMEACR, a Multimodal Memory-Enhanced Agent Collaboration framework for recommendation. MMEACR introduces a dual-track memory architecture that separates interpretable agent reasoning from fine-grained multimodal matching. In the reasoning track, collaborative User and Item Memory Agents maintain persistent multimodal memories and update them through an attribute-guided reinforcement-and-reflection mechanism. In the matching track, a decoupled multi-modal embedding memory is built from raw interaction narratives and item images to preserve detailed cross-modal signals beyond structured memory updates. The two tracks are integrated through weighted Reciprocal Rank Fusion to produce robust and interpretable rankings. Experiments on three real-world domains show that MMEACR achieves strong overall performance against competitive LLM-based and agent-based baselines, with notable gains in visually grounded recommendation scenarios.

## 2. 论文提出的方法论

### 核心思想
- **双轨记忆架构**：将可解释的智能体推理（推理轨道）与细粒度多模态匹配（匹配轨道）分离，最后通过加权倒数排名融合（RRF）整合。
- **推理轨道**：用户记忆智能体（UA）和物品记忆智能体（IA）维护持续更新的多模态记忆（文本+视觉描述），通过属性引导的强化-反思机制迭代进化。
- **匹配轨道**：解耦的多模态嵌入记忆，保留原始交互叙述和物品图像中的密集跨模态信号，避免结构化记忆过滤丢失细微信息。

### 关键技术细节

#### 多模态智能体记忆初始化
- 对每个物品：使用多模态LLM（如GPT-4o）生成图像描述 \( D_i = \text{MLLM}(T_i, V_i) \)，结合标题形成初始物品记忆 \( M_i^{(0)} \)。
- 对每个用户：用领域特定模板初始化用户记忆 \( M_u^{(0)} \)。

#### 属性引导的记忆进化（核心）
- 每个交互步骤构造三元组 \( (u, i^+, i^-) \)，LLM选择匹配项并生成理由。
- 提取偏好相关属性：从预定义语义属性空间 \( A \) 中，基于当前记忆、对比项和理由，生成结构化偏好信号 \( z^{(t)} \)。
- 若选择正确 → **强化**（reinforce）当前偏好；若错误 → **反思**（reflect）修正记忆。
- 用户记忆更新公式：
\[
M_u^{(t+1)} = 
\begin{cases}
f_{\text{rein}}(x_u^{(t)}), & y^{(t)}=1 \\
f_{\text{refl}}(x_u^{(t)}), & y^{(t)}=0
\end{cases}
\]
- 物品记忆异步更新：正物品 \( M_{i^+}^{(t+1)} \) 和负物品 \( M_{i^-}^{(t+1)} \) 分别调整。
- 同时维护原始用户/物品叙述 \( H_u, H_i \)（未过滤的交互历史）。

#### 多模态嵌入记忆（匹配轨道）
- **物品嵌入**：使用预训练多模态嵌入模型MEM，对每个物品i的标题和所有图像取平均得到 \( e_i \)。
- **用户嵌入**：基于用户原始叙述 \( H_u \) 和历史上偏好物品的图像，平均得到 \( e_u \)。
- **嵌入排序**：余弦相似度 \( s_{\text{emb}}(u,i) \)。

#### 混合排名（RRF）
- 推理排名 \( \pi_{\text{des}} \) 和嵌入排名 \( \pi_{\text{emb}} \) 各自计算得分：
\[
S_{\text{des}}(i) = \frac{1}{k_{\text{des}}+\text{rank}_{\pi_{\text{des}}}(i)},\quad S_{\text{emb}}(i) = \frac{1}{k_{\text{emb}}+\text{rank}_{\pi_{\text{emb}}}(i)}
\]
- 最终融合得分：\( S_{\text{RRF}}(i) = w_{\text{des}} S_{\text{des}}(i) + w_{\text{emb}} S_{\text{emb}}(i) \)
- 按降序排列得到推荐列表。

## 7. 优点

1. **双轨架构设计新颖**：将可解释推理与密集跨模态匹配分离，避免LLM推理被冗长多模态信息干扰。
2. **属性引导的记忆进化**：通过预定义属性空间提取关键信号，强化正确偏好、纠正错误，有效抑制噪声和漂移。
3. **视觉-文本深度融合**：利用多模态LLM生成图像描述，使智能体“看见”并理解视觉属性，超越纯文本方法。
4. **可解释性强**：推理轨道产生自然语言理由和记忆，用户可理解推荐逻辑。
5. **效率优化**：精简记忆设计和并行批处理，推理速度快于传统智能体方法AgentCF。
