## 摘要
在多智能体追击-逃避博弈（PEG）中计算纳什均衡策略极具挑战性，因为联合状态和动作空间随智能体数量呈指数增长。现有方法要么依赖离线均衡近似（在执行过程中可能缺乏适应性），要么采用在线规划方法（但面临巨大的分支因子）。本文提出了一种混合框架——原始引导树搜索（PGTS），该框架将离线精确纳什均衡计算与在线树搜索相结合：PGTS首先离线求解一组规模较小、易于处理的子博弈；在部署时，PGTS在每个时间步执行在线树搜索，利用最优子博弈策略和值函数来引导树扩展并估计叶节点值。在包括真实世界网络在内的多种图拓扑结构上的大量实验表明，PGTS显著优于最先进的基于学习和启发式的基线方法，同时保持了对抗对手的鲁棒性能。

## Abstract
Computing Nash equilibrium policies in multi-agent Pursuit-Evasion games (PEG) is challenging due to the exponential growth of the joint state and action spaces with the number of agents. Existing approaches either rely on offline equilibrium approximations, which may lack adaptability during execution, or online planning methods, which suffer from large branching factors. In this work, we propose Primitive-Guided Tree Search (PGTS), a hybrid framework that integrates offline exact Nash equilibrium computation with online tree search: PGTS first solves a collection of smaller, tractable sub-games offline; at deployment, PGTS performs online tree search at each time step, using the optimal sub-game policies and value functions to guide tree expansion and estimate leaf-node values. Extensive experiments on varied graph topologies, including real-world networks, demonstrate that PGTS significantly outperforms state-of-the-art learning and heuristic baselines, while maintaining robust performance against adversaries.

### 论文提出的方法论
- **核心思想**：Primitive-Guided Tree Search（PGTS）将离线精确纳什均衡计算与在线树搜索结合。离线阶段求解一组规模小、可处理的“原始子博弈”（如1v1、2v1），获得最优策略和值函数；在线阶段在每步执行有限深度的树搜索，用原始策略引导动作扩展，用原始值函数估计叶节点值。
- **关键技术细节**：
  - **离线阶段**：采用Shapley值迭代（式(3)-(5)）精确求解1v1和2v1子博弈的纳什均衡，缓存策略 $\pi^*_{1v1}, \pi^*_{2v1}$ 和价值函数 $V^*_{1v1}, V^*_{2v1}$。
  - **原始引导扩展**：对每个蓝方智能体，找出其最近的 $k_r$ 个红方智能体，利用 $\pi^*$ 采样局部联合动作（1v1和2v1），再补充最短路径和随机动作，形成缩减动作集 $\tilde{\mathcal{A}}$，控制分支因子。
  - **叶节点值估计**：将叶节点状态分解为多个1v1和2v1子问题，通过两阶段匈牙利算法（先2v1再1v1）最大化原始值之和（式(9)），作为叶节点值的替代估计。
  - **树搜索选择算子**：支持两种算法——**Regret Matching (RM)**（基于累积遗憾的混合策略选择，Algorithm 1）和**Decoupled UCT (DUCT)**（独立UCB选择，Algorithm 2）。
  - **整体流程**：Algorithm 3 描述了每个时间步的M次模拟：选择→扩展（含原始引导）→叶值估计→回传，最终从根节点提取并执行策略。

### 论文主要结论与发现
- **PGTS显著优于所有基线**：在WCU和ER上，PGTS-RM和PGTS-DUCT在所有四个图上一致获得最高值，尤其在最困难的Scotland Yard和Atlanta图上，优势更明显。
- **学习基线过拟合**：MT-PSRO和NSGZero在SP-WCU上表现良好，但对非最短路径的对抗Evader（WCU）性能大幅下降，说明其策略对训练分布敏感。
- **原始引导扩展和2v1原始关键**：消融显示去掉两者均导致WCU明显降低，尤其在长视野图上。
- **分解基线有效但有限**：Decomposition方法优于启发式，但远落后于PGTS，说明团队协调需要在线搜索。

### 优点
- **混合框架创新性**：巧妙结合离线精确解和在线搜索，避免了学习估值函数导致的数据依赖过拟合问题。
- **计算效率高**：通过原始引导扩展大幅缩减分支因子，叶节点使用分解估值避免蒙特卡洛 rollout，每步仅需0.5秒决策时间。
- **鲁棒性强**：不依赖特定对手分布，在对抗最坏情况Evader时仍保持高回报。
- **模块化设计**：两种选择算子（RM/DUCT）可插拔，便于适配不同场景；离线原始求解可复用。

### 不足与局限
- **实验覆盖不足**：主要聚焦Nv1场景，对NvM（如多对多）仅声称可扩展但未提供定量结果；未在完全随机的转移函数上验证。
- **离线计算成本高**：2v1子博弈在大图上（如Scotland Yard）需6.5小时，扩展到更大规模或更多原始类型（如3v1）可能不可行。
- **依赖手工原始类型**：需事先设计原始子博弈的规模（1v1,2v1），缺乏自动发现交互模式的机制。
- **缺乏统计可靠性度量**：未报告多次独立实验的标准差或置信区间，无法判断性能差异的显著性。
- **假设完全可观测与确定性**：未讨论部分可观测或随机转移的适配，限制了在实际复杂环境中的应用。
