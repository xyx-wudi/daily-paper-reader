## 摘要
多智能体强化学习系统在平衡全局协调与局部执行时面临跨时间尺度的基本挑战。本文介绍耦合层次化多智能体系统（CHMAS），一种新颖框架，将多智能体决策分解为集中式策略规划和分布式战术执行，并具有双向信息流。策略层整合所有智能体状态与独占的全局环境状态，每T个时间步生成引导动作，而战术智能体执行由策略引导和局部邻居观测增强的分布式策略。与现有的单向控制层次化方法不同，CHMAS建立了反馈机制，其中累积的战术奖励通过耦合系数λ影响策略目标，确保策略计划基于战术可行性。为了解决层次化学习中的非平稳性，我们提出异步更新协议，其中策略参数每N_f个战术回合更新一次，允许战术策略在策略变化之间收敛到准平稳点。我们提出捕捉完整系统动态的一般双层公式和可处理的加性近似，以便进行严格分析。理论分析证明，在标准假设下，该异步方案经过K次策略更新后，策略层达到O(log K/√K)的收敛速度。在多智能体觅食领域的实验验证展示了空间划分探索策略的成功学习，尽管存在层次化耦合，两层均稳定收敛。

## Abstract
Multi-agent reinforcement learning (MARL) systems face fundamental   challenges in balancing global coordination with local execution   across different temporal scales. This paper introduces the Coupled   Hierarchical Multi-Agent System (CHMAS), a novel framework that   decomposes multi-agent decision-making into centralized strategic   planning and distributed tactical execution with bidirectional   information flow. The strategic layer integrates all agents' states   with an exclusive global environmental state to generate guidance   actions every $T$ timesteps, while tactical agents execute   distributed policies augmented by strategic guidance and local   neighborhood observations. Unlike existing hierarchical approaches   with unidirectional control, CHMAS establishes a feedback mechanism   where accumulated tactical rewards influence strategic objectives   through a coupling coefficient $λ$, ensuring strategic plans   remain grounded in tactical feasibility. To address the   non-stationarity inherent in hierarchical learning, we propose an   asynchronous update protocol where strategic parameters update every   $N_f$ tactical episodes, allowing tactical policies to converge to   quasi-stationary points between strategic changes. We present both a   general bi-level formulation capturing full system dynamics and a   tractable additive approximation enabling rigorous analysis.   Theoretical analysis proves that this asynchronous scheme achieves   $\mathcal{O}(\log K/\sqrt{K})$ convergence for the strategic layer   after $K$ strategic updates under standard assumptions. Experimental   validation in a multi-agent foraging domain demonstrates successful   learning of spatially partitioned exploration strategies, with both   layers converging stably despite hierarchical coupling.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：多智能体强化学习（MARL）在复杂环境中面临全局协调与局部执行之间的时空尺度平衡挑战。现有方法存在两难：纯集中式方法因联合状态-动作空间呈指数增长而难以扩展；纯分布式方法虽然可扩展，但牺牲了长程目标的协调质量。此外，多智能体系统天然具有多时间尺度特性（如战略决策需要全局信息并在较长时间尺度上制定，战术反应需要局部信息并快速执行），但现有层次化MARL方法（如Feudal Networks、选项学习等）多采用单向控制（高层决定低层），缺乏低层向高层的反馈，导致战略规划可能脱离战术可行性。

- **核心问题**：如何在层次化MARL中建立**双向信息流**——既让战略层利用全局信息引导战术层，又让战术层的执行效果反馈影响战略层目标，同时管理由此带来的非平稳学习动态。

- **整体含义**：本文提出的CHMAS框架提供了一种新的架构范式，通过**耦合系数λ**和**异步更新协议**，使战略层与战术层在学习过程中双向交互且稳定收敛，为多智能体系统中的时空多尺度协调问题提供了理论上有保证、实践上可行的解决方案。

## 2. 方法论：核心思想、关键技术细节、公式与算法流程

### 2.1 核心思想
- 将多智能体决策分解为**集中式战略规划**（Strategic Layer）和**分布式战术执行**（Tactical Layer）。
- 战略层每 `T` 个时间步（一个epoch）选择一个引导动作 `astr_i,k` 广播给每个战术智能体，该引导在 `[kT, (k+1)T)` 内保持固定。
- 战术层每个智能体基于局部观测（自身状态、邻居状态、战略引导）执行分布式策略，并收集奖励。
- **双向耦合**：向下耦合（战略→战术）通过引导改变战术观测和奖励；向上耦合（战术→战略）通过累积战术奖励（乘以系数λ）修改战略奖励函数，使战略目标反映战术可行性。
- 为解决非平稳性，采用**异步更新协议**：战略参数每 `N_f` 个战术回合更新一次（使用衰减步长 `ηstr_k = α/√k`），战术参数在每个回合都更新（步长 `ηtac = β/√K`），且战术层在两次战略更新之间有足够时间收敛到准平稳点。

### 2.2 关键技术细节

#### 战略MDP定义
- 状态：`sstr_k = (senv_k, s1_k, ..., sN_k)`，其中 `senv` 是全局环境状态（如网格覆盖情况），仅战略层可访问。
- 动作：为每个智能体生成个性化引导 `astr_i,k`。
- 奖励：`Rstr(sstr_k, astr_k) = Rglobal(senv_k, astr_k) + λ * Σ_{t=i}^{...} Σ_i rtac_i,t`。
- 战略策略 `πstr: Sstr → P(Astr)` 为全局协调提供指导。

#### 战术多智能体MDP
- 每个智能体观测：`otac_i,t = φ(si,t, {sj,t}_j∈Ni, astr_i,k)`，结合局部状态、邻居状态和战略引导。
- 过渡概率可分解为图结构上的局部依赖。
- 每个智能体有自己的奖励 `rtac_i,t`，鼓励局部优化和邻居协调。

#### 双向耦合与信息不对称
- 向下耦合：引导影响观测和奖励；向上耦合：累积战术奖励影响战略奖励（λ控制强度）。
- 战略层拥有全局状态访问权，战术层仅能访问邻居信息，引导作为信息瓶颈连接两层。

#### 加性近似（Additive Approximation）
- 由于精确双层优化难以处理，作者提出加性近似：`Jglobal = Jstr(θstr) + Σ_i Jtac_i(θtac_i)`，忽略显式交叉依赖，但通过异步更新调度隐含处理。近似误差由战术最优性间隙 `δk` 控制，随学习过程消失。
- 这使得梯度可分离，更新可视为联合参数空间的坐标步骤，便于使用两时间尺度随机逼近进行收敛分析。

### 2.3 算法流程（Asynchronous Hierarchical Policy Gradient, AHPG）

```
初始化 θstr_1, {θtac_i,0}
超参数 T, Nf, α, ηtac, λ
e=0, k=1
for episode e=1,2,... do
     if (e mod Nf == 1) then  // 战略规划阶段
         观测完整状态 sstr_k = (senv_k, sk)
         生成 astr_k ~ πstr(sstr_k; θstr_k)
         广播 astr_i,k 给每个智能体 i
     end if
     // 战术执行阶段（T个时间步）
     for t=0 to T-1 do
         for each agent i in parallel do
             构建观测 otac_i,t = φ(si,t, {sj,t}_j∈Ni, astr_i,k)
             采样动作 atac_i,t ~ πtac_i(otac_i,t; θtac_i,e)
             存储经验
         end for
     end for
     // 战术策略更新（每个回合）
     for each agent i in parallel do
         计算梯度 gtac_i = ∇ Jtac_i
         更新 θtac_i,e+1 = θtac_i,e + ηtac * gtac_i
     end for
     // 战略更新（每 Nf 个回合一次）
     if (e mod Nf == 0) then
         计算累积战术奖励 Rtac_k = Σ_t Σ_i rtac_i,t
         计算战略奖励 rstr_k = Rglobal + λ * Rtac_k
         计算战略梯度 gstr = ∇ Jstr
         更新 θstr_k+1 = θstr_k + (α/√k) * gstr
         k = k+1
     end if
end for
```

### 2.4 理论收敛结果
- 在标准假设（L-光滑性、PL条件、战略梯度偏差有界等）下，战略层经过K次更新后，**最小梯度范数期望收敛率为 `O(log K/√K)`**。
- 战术层平均梯度范数期望收敛率为 `O(1/√K)`。
- 证明关键：通过偏差界 `||bk|| = O(1/√k)` 和衰减步长处理非平稳性。
