## 摘要
自我演化多智能体系统（MAS）已成为一种有前景的途径，使LLM智能体能够从经验中持续改进，其基础是持久记忆。然而，现有设计几乎都采用跨智能体共享的中心化存储库，这带来了通信和协调开销，引发隐私问题，并削弱了智能体的多样性。我们提出DecentMem，一种去中心化记忆框架，其中每个智能体维护自己的双池记忆——一个由过往轨迹整合而成的利用池，以及一个由LLM为未见上下文生成的候选探索池。这两个池基于来自LLM作为评判者的阶段性反馈进行在线重新加权。理论上，我们证明该设计保证了解决方案空间的全局可达性，并实现了$O(\log T)$的累积遗憾，匹配随机赌博机下界（常数项不计）。在实践中，跨越三个MAS框架（AutoGen、DyLAN、AgentNet）、三种Qwen3骨干网络（4B/8B/14B）、两种Gemma4骨干网络（E2B/E4B）以及涵盖数学、代码、问答和具身任务的五个基准测试，DecentMem相比最强的中心化记忆基线，平均准确率提升高达23.8%，相比无记忆基线提升高达52.5%，同时将令牌使用量减少高达49%。

## Abstract
Self-evolving multi-agent systems (MAS) have emerged as a promising route to LLM agents that continually improve from experience, with persistent memory at their foundation. However, existing designs almost exclusively adopt a centralized repository shared across agents, incurring communication and coordination overhead, raising privacy concerns, and collapsing agent diversity. We propose DecentMem, a decentralized memory framework in which each agent maintains its own dual-pool memory -- an exploitation pool of consolidated past trajectories and an exploration pool of LLM-generated candidates for unseen contexts. The two pools are reweighted online based on stage-wise feedback from an LLM-as-a-judge. Theoretically, we prove that this design guarantees global reachability of the solution space and achieves $O(\log T)$ cumulative regret, matching the stochastic bandit lower bound up to constants. In practice, across three MAS frameworks (AutoGen, DyLAN, AgentNet), three Qwen3 backbones (4B/8B/14B), two Gemma4 backbones (E2B/E4B) and five benchmarks spanning math, code, QA, and embodied tasks, DecentMem improves average accuracy by up to 23.8% over the strongest centralized memory baseline and by up to 52.5% over the no-memory baseline, while reducing token usage by up to 49%.

## 1. 核心问题与整体含义（研究动机和背景）

现有 LLM 驱动的多智能体系统（MAS）通常依赖**集中式记忆库**——所有智能体共享一个全局存储库。这种设计存在严重缺陷：
- **通信与协调开销**：全局访问导致高延迟和同步成本。
- **隐私风险**：所有智能体的经验暴露在同一个存储库中。
- **智能体多样性丧失**：共享记忆使行为同质化，削弱了角色互补的专业化分工，破坏了多智能体协作的核心价值。

因此，作者提出应转向**去中心化记忆**架构，让每个智能体独立维护自己的记忆，既保留多样性，又提升效率和隐私。

## 2. 方法论：核心思想、关键技术细节

### 2.1 核心思想

**DecentMem** 是一个去中心化双池记忆框架，每个智能体 `a_m` 维护两个私有记忆池：
- **利用池（E-pool）**：存储已整合的过往成功轨迹（含协作路径和动作注释）。
- **探索池（X-pool）**：由 LLM 为未见过上下文生成的候选记忆（新策略）。

两个池通过**在线路由器**动态平衡，利用 LLM-as-a-Judge 的阶段性反馈调整选择概率，使智能体自动学会何时利用经验、何时探索新策略。

### 2.2 关键技术细节

- **记忆表示**：每条记忆 `z = (ξ, r⋆)`，其中 `ξ` 为上下文原型，`r⋆` 包含动作轨迹和自我注释。
- **在线路由**：按权重 `w_{E}, w_{X}` 决定当前选择 E-pool 或 X-pool 的概率 `α = w_E / (w_E + w_X)`。
- **利用检索**：若选 E-pool，按语义相似度 `sim(e(x), e(ξ))` 检索 Top-K 记忆（设阈值 τ），低于阈值则回退到 X-pool。
- **探索生成**：若选 X-pool，LLM 直接生成新记忆用于当前上下文。
- **更新规则**：基于阶段评分 ∆t（当前阶段比上一阶段更好则 +1，否则 0），动态调整 `w_E` 和 `w_X`（X 池权重固定为 1），任务结束后将 X-pool 全部合并到 E-pool。
- **理论保证**：
  - **定理 1（全局可达性）**：在 X-pool 权重恒正且 LLM 先验满支撑下，搜索过程 Markov 链不可约且非周期，保证不会陷入局部最优。
  - **定理 2（对数遗憾）**：在线路由策略满足 `E[R(T)] = O(log T)`，匹配随机赌博机下界 Ω(log T)，达到阶最优。
